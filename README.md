# CashTrack – Expense Tracker

CashTrack is a simple web-based expense tracking application built using **HTML, CSS, and JavaScript**.  
It allows users to set their salary, add expenses, view transaction history, and automatically calculate the remaining balance.

This project was created as a beginner-friendly frontend project to practice **JavaScript DOM manipulation, event handling, and UI design**.


## Features

- Set and edit monthly salary
- Add expenses with description and amount
- Automatic balance calculation
- Transaction history display
- Delete transactions
- Download expense report as PDF
- Clean and simple user interface

---

## Tech Stack

- **HTML** – Structure of the application  
- **CSS** – Styling and layout  
- **JavaScript** – Logic, DOM manipulation, and functionality
